Bootstrap Example

Your final project for this term is to create a 4 page website using the Bootstrap framework. The site is for a fictitious sports league. You choose the sport and the name of the league. Choose a sport you like to watch, a sport you play, or a sport that you wish existed. I hear that Competitive Full-Contact Tiddlywinks is gaining popularity. Layout - The layout will be nearly the same on each page. The header, navbar, and footer should be the same on every page. The layout must be responsive. However, since we are using Bootstrap, that is going to happen almost automatically. Use the CSS classes that are built into Bootstrap for things like headers, navbars, columns, rows, buttons, etc. The purpose of this project is to use images that are royalty free and the Bootstrap framework to minimize the amount of work that you will do. No need to reinvent anything. Use the tools provided with Bootstrap to see how quickly you can build a site with custom images and colours.
